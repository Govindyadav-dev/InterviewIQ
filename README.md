# 🚀 InterviewIQ.AI

An AI-powered SaaS-based interview preparation platform built using the MERN stack.  
Users can upload their resume, select a job role, attend AI-generated mock interviews, receive detailed feedback, and track their interview history.

---

# ✨ Features

- 🔐 User Authentication (Login / Signup / Logout)
- 📄 Resume Upload System
- 🤖 AI-Powered Interview Question Generation
- 🎯 Role-Based Personalized Interviews
- 📊 AI Answer Evaluation & Feedback
- 📝 Detailed Interview Reports
- 📥 Download Report as PDF
- 🕒 Interview History Tracking
- 💳 Razorpay Payment Integration
- 🪙 Credit-Based Interview System
- ⚡ Responsive Modern UI
- 🔄 Redux State Management

#🔄 Application Flow

.User signs up or logs in
.User uploads resume and selects target role
.AI generates role-based interview questions
.User answers questions one by one
.AI evaluates answers and generates feedback
.Final performance report is created
.User can download report as PDF
.Interview history is stored for future tracking

##🧠 AI Workflow
.Resume is processed on the backend
.Resume data + selected role are sent to OpenRouter API
.AI dynamically generates interview questions
.AI evaluates answers based on:
.Relevance
.Clarity
.Technical Accuracy
.Communication Quality
